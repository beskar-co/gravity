# Gravity

Gravity is a WIP design system to power experiences from Beskar Labs.

[![npm version](https://badge.fury.io/js/@beskar-labs%2Fgravity.svg)](https://badge.fury.io/js/@beskar-labs%2Fgravity)

## Installation

Install the package with your package manager of choice.

```sh
yarn add @beskar-labs/gravity
```

Add the relevant providers and components to your application:

```tsx
import { TooltipProvider } from '@beskar-labs/gravity/tooltip';
import { Toaster } from '@beskar-labs/gravity/toast';
import type { ReactNode } from 'react';

const RootLayout: FC<{
  children: ReactNode;
}> = () => (
  <>
    <TooltipProvider>{children}</TooltipProvider>
    <Toaster />
  </>
);

export default RootLayout;
```

Use the components in your application:

```tsx
import { Button } from '@beskar-labs/gravity/button';

const MyComponent: FC = () => (
  <Button onClick={() => alert('Hello world!')}>Click me!</Button>
);
```
